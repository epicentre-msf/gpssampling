# Theme function for customizing Bootstrap themes

This function allows customization of Bootstrap themes by adjusting
various elements such as fonts, colors, buttons, and cards.

## Usage

``` r
theme(preview = FALSE)
```

## Arguments

- preview:

  Logical indicating whether to preview the theme changes (default is
  FALSE)

## Value

A customized Bootstrap theme object

## Examples

``` r
theme()
#> Error in theme(): could not find function "theme"
```
