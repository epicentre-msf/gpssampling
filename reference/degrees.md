# Convert radians to degrees

This function takes an angle in radians and converts it to degrees.

## Usage

``` r
degrees(angle)
```

## Arguments

- angle:

  The angle in radians.

## Value

The angle in degrees.

## Examples

``` r
degrees(pi / 2L)
#> Error in degrees(pi/2L): could not find function "degrees"
# Output: 90
```
