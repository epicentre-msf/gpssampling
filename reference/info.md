# bash alias for [`sessionInfo`](https://rdrr.io/r/utils/sessionInfo.html)

bash alias for [`sessionInfo`](https://rdrr.io/r/utils/sessionInfo.html)

## Usage

``` r
info(package = NULL)
```

## Arguments

- package:

  a character vector naming installed packages, or `NULL` (the default)
  meaning all attached packages.
