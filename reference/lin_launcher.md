# Create a launcher and updater for Linux (.sh)

Create a launcher and updater for Linux (.sh)

## Usage

``` r
lin_launcher()
```

## Details

On Linux a file named 'launch.sh' and one named 'update.sh' will be put
on the desktop. Double-click the file to launch the specified app app or
update app to the latest version

## Examples

``` r
if (FALSE) { # \dontrun{
lin_launcher()
} # }
```
