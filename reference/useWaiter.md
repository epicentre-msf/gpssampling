# Use Waiter and Waitress in Shiny app

This function integrates Waiter and Waitress into a Shiny application.

## Usage

``` r
useWaiter(particle = FALSE)
```

## Value

A tag list containing Waiter and Waitress components for a Shiny app.

## Examples

``` r
useWaiter()
#> Error in useWaiter(): could not find function "useWaiter"
```
