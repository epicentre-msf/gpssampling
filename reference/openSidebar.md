# Open the Sidebar by ID

Open the Sidebar by ID

## Usage

``` r
openSidebar(map, id)
```

## Arguments

- map:

  A leaflet map widget

- id:

  The id of the
  [`sidebar_pane`](https://epicentre-msf.github.io/gpssampling/reference/sidebar_pane.md)
  to open

## Value

the new `map` object
