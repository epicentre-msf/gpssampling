# User functions :

Make the package more user friendly (you need to use this function into
the ui part of your shinny Application)

## Usage

``` r
useGoogle(apikey = Sys.getenv("MAPS_API_KEY_GOOGLE"))
```

## Arguments

- apikey:

  the api-key you want to use for your application (you need to use a
  valid key)

## Value

a tags\$head that will be usefull to load the references js/css
