# Create a launcher on the desktop for Windows (.bat), Mac (.command), or Linux (.sh)

Create a launcher on the desktop for Windows (.bat), Mac (.command), or
Linux (.sh)

## Usage

``` r
launcher()
```

## Details

On Windows/Mac/Linux a file named launch.bat/launch.command/launch.sh
will be put on the desktop. Double-click the file to launch the
specified app app

## Examples

``` r
if (FALSE) { # \dontrun{
launcher()
} # }
```
