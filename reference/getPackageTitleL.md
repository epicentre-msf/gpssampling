# Get Package Title in Lowercase

This function returns the title of a package in lowercase.

## Usage

``` r
getPackageTitleL()
```

## Value

The package title in lowercase.

## Details

This function requires the following packages to be installed: `tools`.

## Examples

``` r
getPackageTitleL()
#> Error in getPackageTitleL(): could not find function "getPackageTitleL"
## 'my package'
```
