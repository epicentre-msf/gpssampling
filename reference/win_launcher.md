# Create a launcher and updater for Windows (.bat)

Create a launcher and updater for Windows (.bat)

## Usage

``` r
win_launcher()
```

## Details

On Windows a file named 'launch.bat' and one named 'update.bat' will be
put on the desktop. Double-click the file to launch the specified app
app or update app to the latest version

## Examples

``` r
if (FALSE) { # \dontrun{
win_launcher()
} # }
```
