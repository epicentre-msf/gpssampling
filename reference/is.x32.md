# Check if the R platform is 32-bit.

This function checks if the R platform is 32-bit by comparing the value
of the 'r_arch' component of the .Platform list to 'x32'.

## Usage

``` r
is.x32()
```

## Value

`TRUE` if the R platform is 32-bit, `FALSE` otherwise.

## Examples

``` r
is.x32()
#> Error in is.x32(): could not find function "is.x32"
```
