# Prompt the user for input and return the entered value.

Prompt the user for input and return the entered value.

## Usage

``` r
ask(prompt)
```

## Arguments

- prompt:

  A character string specifying the prompt to display.

## Value

A character string with the user input.

## Examples

``` r
ask('Enter your name: ')
#> Error in ask("Enter your name: "): could not find function "ask"
ask('Enter a number: ')
#> Error in ask("Enter a number: "): could not find function "ask"
```
